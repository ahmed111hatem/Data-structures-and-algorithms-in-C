Welcome to this repository! Here, we will learn how to implement data structures and algorithms using the C programming language.

We will cover the implementation and use of:

- Trees
- Graphs

#### 🔍 Searching Algorithms
- Linear Search
- Binary Search

- Two Pointers Method
- Sliding Window
- Dynamic Programming
- Backtracking
- Greedy Algorithms
